Detailed Description

Delegate for handling I/O

These functions are events that get fired when saving/loading a space, they enable you to use a specific id system (if you choose) as well as record any UI specific data at time of writing; the reading events will let you hook up your UI when an object has just been read, but not added to the space yet.
