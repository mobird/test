Detailed Description

CC3VertexLocationsBoundingBoxVolume is a type of CC3NodeBoundingBoxVolume specialized for use with CC3VertexArrayMesh and CC3VertexLocations.

The value of the boundingBox property is automatically calculated from the vertex location data by the buildVolume method of this instance.
